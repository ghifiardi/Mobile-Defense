**Seismic Sentry** is a web-based anti-theft alarm that turns your device into a motion-sensitive security system.

## Features
-   **Motion Detection**: Uses device accelerometer/gyroscope.
-   **Visual Alarm**: Flashing red screen on trigger.
-   **Audio Siren**: High-frequency synthesized siren.
-   **Zero Install**: Works directly in the browser.

## How to Use
1.  Open the app on your phone.
2.  Tap **ARM SYSTEM**.
3.  Grant motion sensor permissions (iOS 13+).
4.  Place device on the object you want to guard.
5.  Wait 3 seconds for arming.
6.  **Protected!** Any movement will trigger the alarm.

## Deployment
This project is designed to be hosted on **GitHub Pages**.
1.  Upload files to a GitHub repository.
2.  Go to Settings > Pages.
3.  Select `main` branch and save.
